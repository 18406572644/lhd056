## 1. 产品概述

室内设计平面图编辑器是一款基于Canvas的纯前端Web应用，为用户提供直观的室内平面设计工具。用户可以通过拖拽、绘制等操作快速创建房间布局、添加家具、标注尺寸，并导出设计成果。

- **核心价值**：降低室内设计门槛，让普通用户也能快速进行空间规划和家具布局
- **目标用户**：室内设计师、装修业主、房产经纪人、空间规划爱好者
- **市场定位**：轻量级、零安装、即开即用的在线平面设计工具

## 2. 核心功能

### 2.1 用户角色

本产品无需用户注册，所有功能对访客开放，数据存储在浏览器本地。

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 使用全部编辑功能、导出PNG、本地撤销重做 |

### 2.2 功能模块

1. **画布区域**：全屏Canvas绘制区域，支持网格背景、缩放、平移
2. **左侧工具栏**：选择、墙壁、房间、家具、尺寸、文字六种编辑工具
3. **右侧家具库**：分类展示各类家具，支持拖拽到画布
4. **底部状态栏**：实时显示光标坐标、缩放比例、选中物体信息
5. **操作辅助**：撤销重做、删除、网格吸附、旋转控制
6. **导出功能**：将当前设计导出为PNG图片

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主编辑页 | 画布区域 | 全屏Canvas，网格背景(10cm×10cm)，支持分层绘制 |
| 主编辑页 | 左侧工具栏 | 六种工具切换，选中高亮，悬停提示 |
| 主编辑页 | 右侧家具库 | 分类标签切换(客厅/卧室/厨房/卫浴)，家具卡片列表，拖拽放置 |
| 主编辑页 | 底部状态栏 | 实时坐标(米)、缩放比例、选中物体详情 |
| 主编辑页 | 导出功能 | 一键导出PNG，保留当前画布内容 |

## 3. 核心流程

```mermaid
flowchart LR
    A["进入应用"] --> B["选择工具<br/>(墙壁/房间/家具/尺寸/文字)"]
    B --> C["在画布操作<br/>(绘制/拖拽/点击)"]
    C --> D["物体创建/修改"]
    D --> E["选择工具选中物体"]
    E --> F["调整操作<br/>(移动/旋转/缩放/删除)"]
    F --> G["撤销/重做(可选)"]
    G --> H["导出PNG(可选)"]
```

**主要用户流程说明**：
1. 用户打开应用，默认进入选择工具
2. 用户可选择墙壁工具绘制单墙，或房间工具快速创建矩形房间
3. 用户从右侧家具库拖拽家具到画布，家具自动吸附网格
4. 使用选择工具选中物体后，可通过8个控制点调整尺寸，通过顶部手柄旋转
5. 使用尺寸工具点击两点可标注距离，文字工具添加文字说明
6. 可随时通过Ctrl+Z/Ctrl+Y撤销重做，Delete键删除选中物体
7. 完成设计后点击导出按钮保存为PNG图片

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：白底网格(#ffffff) + 深灰墙体(#333333) + 蓝色选中(#3b82f6) + 红色标注(#ef4444)
- **辅助色**：浅灰网格线(#e0e0e0) + 稍深灰网格线(#c0c0c0) + 深色工具栏(#1e1e2e)
- **按钮风格**：工具栏按钮为简约线条图标，选中时蓝色高亮背景，悬停时轻微放大
- **字体**：标注文字使用DM Sans，坐标数值使用DM Mono等宽字体
- **布局风格**：三栏布局(左工具栏+中画布+右家具库)+底部状态栏，固定定位
- **图标风格**：简约线性图标，线条粗细一致，风格统一

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|---------|
| 主编辑页 | 画布区域 | 白色背景、灰色网格线、分层绘制内容、选中物体控制点 |
| 主编辑页 | 左侧工具栏 | 深色垂直面板、6个工具按钮、选中高亮、图标+文字提示 |
| 主编辑页 | 右侧家具库 | 白色垂直面板、分类标签栏、家具卡片网格、缩略图+名称 |
| 主编辑页 | 底部状态栏 | 深色水平条、三栏信息展示(坐标/缩放/选中)、等宽字体 |
| 主编辑页 | 导出按钮 | 右上角浮动按钮、蓝色背景、白色图标、悬停阴影 |

### 4.3 响应式

本产品为桌面端优先设计，主要面向大屏设备使用：
- 最小支持宽度：1280px
- 画布区域自适应窗口大小
- 工具栏和家具库为固定宽度(左侧60px，右侧280px)
- 不针对移动端优化，不支持触摸操作

### 4.4 视觉效果

- **家具绘制**：半透明彩色填充 + 深色边框 + 中心简约图标
- **选中效果**：蓝色边框高亮 + 8个白色圆角控制点 + 顶部旋转手柄
- **尺寸标注**：红色细线 + 两端箭头 + 白色背景黑字距离数值
- **墙壁绘制**：深灰色填充，12cm厚度，绘制时显示预览虚线
- **拖拽效果**：家具拖入时半透明预览，释放后吸附网格
- **动画**：工具切换、面板展开、按钮悬停等操作添加150ms过渡动画
